Politique de Confidentialité - Sheeepix Studios

Politique Générale

Bonjour ! Je suis Nathan VALENTIN, développeur des jeux et applications sous la marque Sheeepix Studios. Cette politique explique simplement comment votre vie privée est protégée quand vous utilisez mes applications.

DémiBomb: Mission Tactique

Ce que vous devez savoir

  1. Le jeu en lui-même
    • Vos scores et paramètres sont sauvegardés uniquement sur votre appareil
    • Vous pouvez les effacer en désinstallant l'application
  2. Les publicités
    • J'utilise Google AdMob pour afficher des publicités
    • Les publicités peuvent être désactivées en achetant la version Premium
    • Google AdMob peut collecter des données anonymes pour les publicités
    • Vous pouvez désactiver la personnalisation des pubs dans les paramètres de votre appareil.
  3. Les achats
    • Les achats Premium sont gérés par Apple
    • Je n'ai accès à aucune de vos informations bancaires
    • Le statut Premium est lié à votre compte Apple

Protection des enfants

  • Le jeu est conçu pour tous les âges
  • Je ne collecte aucune information personnelle
  • Les publicités sont adaptées à tout public

Vos droits

  • Vous pouvez demander des informations sur vos données à tout moment
  • Vous pouvez désactiver les publicités personnalisées
  • Vous avez le contrôle total de vos données

Contact

  • LinkedIn
  • Instagram

DémiBomb: Mission Tactique

Guide de jeu

Modes de jeu :

  • Mode Classique : Repérez et marquez toutes les mines
  • Mode Désamorçage : Contre la montre ! Déminez avant la fin du temps imparti
  • Mode Trésor : Trouvez le trésor caché pour gagner (Premium seulement)

Niveaux de difficulté :

  • Recrue : 9x9 cases, 10 mines (1 minute en mode Désamorçage)
  • Sergent : 16x16 cases, 40 mines (3 minutes en mode Désamorçage)
  • Général : 20x20 cases, 99 mines (5 minutes en mode Désamorçage) (Premium seulement)

Problèmes fréquents

Q: J'ai acheté le pass Premium à vie (Vétéran) mais j'ai changé d'appareil iOS. Comment restaurer mon achat ?

R: Accédez à la boutique et utilisez le bouton "Restaurer les achats"

Q: Les publicités ne se chargent pas ?

R: Si vous avez un pass Premium, vous n'avez pas de publicités. Sinon, vérifiez votre connexion Internet et redémarrez l'application
